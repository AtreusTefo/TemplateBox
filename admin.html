Blog Admin

Write and manage blog posts locally, then export the data file and deploy it. Nothing here touches a server: drafts live in this browser's storage until you publish the exported js/blog-data.js through Git or Netlify.

Publish

How publishing works
  1. Save your posts below. They are stored only in this browser.
  2. Click Download js/blog-data.js and replace the existing file at js/blog-data.js in the project.
  3. Commit and push to main (Netlify auto-deploys), or drag the site folder into Netlify.
  4. Visitors now see the updated posts on the live blog.

Formatting reference
  • ## Heading and ### Subheading
  • Blank line starts a new paragraph
  • - item for bullet lists, 1. item for numbered lists
  • > text for a quote
  • **bold**, *italic*, [link text](https://example.com)
  • [image: https://url | alt text] on its own line for an inline image
